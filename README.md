# base

## git cloneしたら

```
# yarn
yarn

# npm
npm install
```

### ローカルで立ち上げるなら

```
# yarn
yarn dev

# npm
npm run dev
```

### ビルドするなら

```
# yarn
yarn build

# npm
npm run build
```

### Lints and fixes files

```
# yarn
yarn lint

# npm
npm run lint
```
